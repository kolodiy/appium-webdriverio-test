# appium-webdriverio-test

Project for experiments with WebdriverIO and Appium

### Prerequisites
* Node v8.11.1 (LTS);
* Appium 1.7.2;
* Latest XCODE to run iOS simulators


### Installing

* Clone project
* Under project root run:
```
 npm i
```

## Running the tests

 * Start Appium server
 * Under project root run:
 ```
  npm run appium
 ```
